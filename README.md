# Usage

## spirals_main.py

| Arguments | Description |
| ----------- | ----------- |
| --net | 'raw' (default), 'raw', 'short' |

| --hid | '10' (default) Number of hidden units |
| --lr | '0.01' (default) Learning Rate |
| --epochs | '100000' (default) Max Training epochs |
